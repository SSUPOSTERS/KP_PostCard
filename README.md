## Korea Post Hackathon, Posters

### Post Card

#### Feature
- 나만을 위한 커스터마이즈 카드 제작
- 본인의 혜택 및 카드 디자인을 사용자 스스로 선택할 수 있는 상품
- White Consumer의 유입 가능 및 소비자의 효율적인 소비 촉진
- 새로운 결재 방법 도입을 통한 사용자 편의성 증대
- 물품을 고른 즉시, 바코드를 통한 결제 방식 제안
